
## Description

Esta apliacion es una demo para aprender NextJs, el objetivo es interactuar con todas las funcionalidades del Framework lo maximo posible.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
